# Carrinho de Compras

Projeto de e-commerce desenvolvido com **JavaScript Vanilla** para praticar manipulação de DOM e lógica de estado.

**Acesse o projeto online:** [Clique aqui para ver o Demo](https://sage-salamander-db8b51.netlify.app/)

##  Funcionalidades
*   **Vitrine Dinâmica:** Renderização de produtos via JavaScript.
*   **Carrinho Inteligente:** Adição e remoção de itens com atualização automática de quantidade.
*   **Cálculo Automático:** O valor total é recalculado a cada interação.
*   **Persistência:** Os itens salvos no carrinho permanecem após atualizar a página (`localStorage`).

##  Tecnologias
*   HTML5
*   CSS3
*   JavaScript (ES6+)
